사람을 생각하는 디자이너, 다양한 분야를 가리지 않고 배우는 웹 개발자입니다.
포토샵, 인디자인, 일러스트를 통해 디자인의 감각을 넓혔고, HTML, CSS, Javascript, React를 중심으로 웹 개발을 익혔습니다.

웹의 Front-end 뿐만 아니라 back-end까지 사용해보면서 협업 할 때 다양한 경험을 바탕으로 원활하게 작업 할 수 있습니다. 현재는 full-stack을 목표로 노력하고 있습니다.

  • 2017.03 - 2020.01 안화고등학교 재학 및 졸업.

저의 활동사항을 소개합니다.

01. 다양한 분야 부터 시작...

(1) C 언어로 프로그래밍 처음 입문 후 넥슨 코딩대회 참가.
(2) Python을 이용해 ComputerVision분야 AI를 주제로 사제동행 사다리 프로젝트 참가.
(3) Python을 이용해 범죄차량 도주경로 예측을 주제로 메이커 대회에 참가.

02. 웹 개발에 뛰어들다.

고려 IT 직업 전문 학교의 웹 컨텐츠 개발을 수료
(1) Photoshop, Indesign, Illustrator 를 중심으로 웹 디자인을 배우고 익힘.
(2) HTML, CSS, Javascript를 사용해 웹 사이트 제작.
(3) React와 nodeJs를 사용해 더욱 다양한 방법, 체계적으로 웹 사이트 개발 및 관리

03. 새로운 도전.

웹 개발에 그치지 않고 어플리케이션 개발에도 관심을 가지게 됨.
(1) React Native를 사용해 어플리케이션을 제작해봄.
